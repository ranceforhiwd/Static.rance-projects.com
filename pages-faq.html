AWS Document Pipeline
What does this system do?

This Serverless system will convert PDF documents into PNG file, then perform an OCR scan and save the text into a DynamoDb Table.

System Diagram

Components

S3 buckets are configured to filter for specific file types (PDF). Data is intentially saved in separate buckets to decouple processed data while converting files.

Docker Containers

  • Convert PDF files into PNGs using PHP & Ghostscript
  • OCR PNG files using Tesseract by Google

Throttle the process using queues containing messages for containers to use as parameters.

Using NOSQL option for scanned text data also leveraging DynamoDb scaling features along with the ability to trigger ohter processes based on record formats.
